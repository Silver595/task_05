# Pranav Portfolio

A modern personal portfolio built with React, Vite, and Tailwind CSS.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Font Awesome
- Google Fonts

## Project Structure

```text
Portfolio_2/
  index.html
  package.json
  public/
    images/
  src/
    App.jsx
    index.css
    main.jsx
  tailwind.config.js
  postcss.config.js
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Notes

Generated folders and logs such as `dist/`, `node_modules/`, and `vite-dev.*.log` are ignored by Git.
